# CCNP ENCOR Guided Lab Packets

このフォルダは、`EVE-NGで構成を組む`、`コンフィグを手打ちする`、`showコマンドで確認する`、`検証結果をZenn記事化する` ことに集中するための学習用Markdown置き場です。

## このフォルダの使い方

1. `ccnp_encor_lab_roadmap.md` を見て、次に学習する章と記事を決める
2. 各章フォルダの `zenn_*.md` を見て、EVE-NGの構成を作る
3. コンフィグを手打ちし、`show` / `ping` / `traceroute` / Wiresharkで検証する
4. 取得した結果だけを、Markdownの `TODO:` 箇所へ貼る
5. そのまま同じ `zenn_*.md` を記事として整える

## ルール

- 実際に取得していない結果は書かない
- 未取得の結果は `TODO: ここに実際の結果を貼る` のまま残す
- 1ラボ1テーマで進める
- 記事は検証ログとして残す
