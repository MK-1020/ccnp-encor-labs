# Lab 01: L2 Forwarding

## Overview

This lab verifies basic Layer 2 forwarding behavior using EVE-NG.

## Topics

- VLAN
- Access port
- Trunk port
- MAC address table
- 802.1Q tag
- ARP broadcast

## Topology

SW1 and SW2 are connected by a trunk link.

- VLAN 10: PC1, PC3
- VLAN 20: PC2, PC4

## Verification

- show vlan brief
- show interfaces trunk
- show mac address-table dynamic
- Wireshark capture on access and trunk links
